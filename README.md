## Инструкция по запуску

Предварительно склонируйте репозиторий
1. **Загрузка зависимостей**
```bash
pip install -r requirements.txt
```

2. **Настройка переменных окружения**
Измените пути с датасетами в **.env**

3. **Запуск скрипта**
```python
python script.py
```
